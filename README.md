# Course of Webpack by EdTeam-@jonmircha

### Clone Repo
```bash
$ git clone <repo>
$ cd courses-webpack
```
### Install Dependency

```bash
yarn install
```

### Start The Development Server

```bash
yarn start
```
Example Runnign on: [http://localhost:8080/](http://localhost:8080/).

### Run Project On Production

```bash
yarn build
```
